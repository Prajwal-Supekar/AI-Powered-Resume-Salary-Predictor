# 💼 AI Resume Salary Predictor

This project predicts salary using Machine Learning.

## 🚀 Features
- User input via Streamlit UI
- Predict salary based on experience, education, job title
- ML model using Linear Regression

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-learn
- Streamlit

## ▶️ Run Project

```bash
streamlit run app.py
